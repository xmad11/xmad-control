# DESIGN TOKENS — ALLOWED USAGE

## Layout Tokens
- --header-height
- --panel-offset-top
- --panel-width
- --panel-radius

## Spacing Tokens
- --space-xs
- --space-sm
- --space-md

## Icon Tokens
- --icon-size-lg
- --icon-size-md

## Rules
- Tokens may be READ
- Tokens may be REFERENCED
- Tokens may NOT be redefined in components
- Tokens may NOT be overridden inline

Tokens are the ONLY source of truth.

Source:
Token-only layouts are the recommended scalable UI pattern
