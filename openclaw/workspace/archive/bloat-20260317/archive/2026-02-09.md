## Technical Details

### Model Configuration Structure
Two different config files with different formats:

1. **Main Config** (`/Volumes/Transcend/openclaw/openclaw.json`):
   ```json
   {
     "model": {
       "primary": "anthropic/glm-4.7"  // With provider prefix
     }
   }
   ```

2. **Agent Models** (`/Volumes/Transcend/openclaw/agents/main/agent/models.json`):
   ```json
   {
     "providers": {
       "anthropic": {
         "models": [
           {
             "id": "glm-4.7",  // WITHOUT provider prefix (already inside anthropic section)
             "name": "GLM-4.7 (via z.ai)"
           }
         ]
       }
     }
   }
   ```

### Why Two Different Formats?
- `openclaw.json`: References models from any provider, needs full `provider/model` format
- `models.json`: Defines models UNDER each provider section, provider is implicit
